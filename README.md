# Yii2 Wechat Module For Backend
 
## Quick Start

* Requirement

  Config [yii2-easy-wechat](https://github.com/max-wen/yii2-easy-wechat) First  

* Installation

```
composer require zacksleo\yii2-wechat

```
* Migration

```
yii migrate/up --migrationPath=@zacksleo/yii2/wechat/migrations

```
